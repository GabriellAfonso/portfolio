Sobre mim

Olá, meu nome é Gabriel, sou um programador focado em desenvolvimento web na area de backend, que é a area que mais me interessa, já que gosto de resolver problemas e criar soluções para situações complexas.

Tenho experiencia em programação com python, e estou me especializando em desenvolvimento backend com o framework Django. Ao longo dos meu projetos, pude aplicar meu conhecimento em python para criar sistemas robustos e de alto desempenho.

Além disso, possuo conhecimentos sólidos em HTML, CSS e JavaScript, o que me permite ter uma visão completa da pilha de desenvolvimento web. Essas habilidades me permitem colaborar de forma eficaz com equipes front-end, garantindo uma integração perfeita entre o backend e o frontend.

Você pode conferir alguns dos meus projetos e contribuições no meu perfil do GitHub e na aba projetos do meu portifólio onde você podera acessar os detalhes de cada projeto onde eu descrevo a estrutura os desafios e as tecnologias usadas em cada projeto.
